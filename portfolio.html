Numeral Base Converter

The numeral base converter is a command line utility I designed to convert numbers between different numeral bases. For example, it can convert numerals between binary, decimal, octal, and hexadecimal but also more unusal bases (like base 3 for example). Like Java's Integer.parseInt() method, it will work with any base between 2 and 16 inclusive. This project started as a lab for my architecture and compilers course at Willamette University, but I've modified it as a CLI application designed to be piped to other CLI apps. Check it out on my Github.

Longest Common DNA Subsequence

The longest common DNA subsequence started as a project in my algorithms course which has since been expanded It can find the longest common subsequence between two DNA sequences, find compliment bases, and convert DNA to Messenger RNA. I've found this project particularily interesting after I started working in biotech.
